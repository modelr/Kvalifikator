# Kvalifikator

Приложение хранит бизнес-данные в Supabase (облако), а файлы сделки (`notes.txt`, вложения) — в локальной папке `Квалификатор`, путь к которой задаётся на каждом рабочем месте отдельно.

## Как это работает
- **Supabase**: общая облачная БД для всех пользователей.
- **Локальная папка**: у каждого пользователя свой путь на ПК.
- Если локальная папка синхронизируется облачным диском (OneDrive/Dropbox/Google Drive и т.п.), файлы сделок будут доступны на других устройствах после синхронизации.

## Первый запуск
1. Запустите приложение.
2. Нажмите **«Изменить путь»** и выберите корневую папку `Квалификатор`.
3. Путь сохранится в профиле пользователя (Electron `userData`) и будет использоваться при создании новых сделок.

## Сборка .exe (Windows installer)
```bash
npm install
npm run dist:win
```

Готовый установщик появится в `dist/`.

## Что попросить у пользователя при установке
На этапе первой настройки в самом приложении пользователь выбирает путь к папке `Квалификатор` через кнопку **«Изменить путь»**. Это заменяет жёстко прописанный путь и делает установку переносимой между компьютерами.
